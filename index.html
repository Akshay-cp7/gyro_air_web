Important Security Notice

When downloading GyroAir PC Client, Windows may show a "Publisher could not be verified" warning or "Windows protected your PC" message from SmartScreen.

This is normal for unsigned open source software. To proceed:

  1. Click "More info" on the SmartScreen warning
  2. Then click "Run anyway" to install
  3. The software is safe - this warning appears because we don't have a paid code signing certificate

Why is GyroAir safe?

  • Open Source: All code is publicly available on GitHub for inspection
  • No Data Collection: Works entirely on your local network
  • No Internet Required: Only connects between your phone and PC
  • Transparent: You can see exactly what the software does
