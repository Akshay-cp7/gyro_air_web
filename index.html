Get Started in 3 Simple Steps

Connect your devices and start controlling your PC wirelessly in under 2 minutes

1

Same Wi-Fi Network

Ensure both your PC and smartphone are connected to the same Wi-Fi network. This creates a secure local connection between your devices.

2

Install PC Client

Download and run the GyroAir PC application. Run as administrator for optimal performance and allow through Windows Firewall when prompted.

3

Start Phone Server

Install the mobile app, enter your PC's IP address, and start the server. Your phone is now a wireless mouse and keyboard!

Frequently Asked Questions

Everything you need to know about GyroAir

Is GyroAir completely free?

Yes! GyroAir is completely free and open-source. There are no hidden fees, premium features, or subscriptions. You get full functionality at zero cost.

Do I need to install anything on my phone?

Yes, you'll need to install the GyroAir mobile app on your Android device. The app is lightweight and provides the interface to control your PC wirelessly.

Why does Windows show a security warning?

Windows SmartScreen flags unsigned software from new developers. GyroAir is safe - you can verify this by checking our open-source code on GitHub. Simply click "More info" then "Run anyway" to install.

What's the range of the wireless connection?

The range depends on your Wi-Fi network coverage. As long as both devices are connected to the same Wi-Fi network, GyroAir will work anywhere within your network's range.

How accurate is the gyroscope control?

The gyroscope control is extremely precise and responsive. Modern smartphones have high-quality gyroscope sensors that provide smooth, accurate cursor movement. You can adjust sensitivity in the app settings.
